## 报告主体 / Report Subject
瑞银(UBS)关于澳大利亚多元化矿业公司2026年3月季度业绩前瞻及成本情景分析报告，涉及公司包括：必和必拓(BHP)、力拓(RIO)、Fortescue Metals(FMG)、美铝(Alcoa Corporation, AAI)、South32(S32)、Mineral Resources(MIN)、Whitehaven Coal(WHC)、Coronado Global Resources(CRN)和Deterra Royalties(DRR)。

## 核心观点 / Key Takeaways
1. 中东冲突导致能源成本上升，可能增加矿业公司运营成本。由于原油价格年内上涨60-70%，柴油价格上涨高达90%，铁矿石矿商的成本敏感性可能在每吨2-3美元之间。
2. 季节性天气因素影响生产，特别是西澳大利亚皮尔巴拉和昆士兰煤炭地区的雨季天气，以及气旋对生产的影响。
3. 各公司将发布2026财年指引，投资者应关注成本指引和评论，以及中东冲突对能源价格的持续影响。
4. 管理层变动和战略调整将成为关注焦点，包括必和必拓新任CEO布兰登·克雷格的任命，以及South32新CEO的战略方向。
5. 锂市场动态值得关注，Mineral Resources正在向POSCO出售锂资产权益，同时Bald Hill矿可能重启，因为锂市场价格已从低谷回升。

## 关键数据 / Key Data
- BHP成本可能增加约2.4美元/吨，RIO增加约2.6美元/吨，FMG增加约3.0美元/吨
- RIO在3月季度因皮尔巴拉气旋损失约800万吨铁矿石产量
- 澳元兑美元年内上涨约6%
- UBS预测BHP铁矿石实现价格约为83美元/吨，比市场共识低3%
- MIN的铝出货量预测比共识低5%

## 投资建议 / Investment Recommendation
报告未明确给出具体的买入/卖出/持有建议，但提供了各公司的季度业绩前瞻和关键关注点。

## 风险提示 / Risk Factors
1. 中东冲突持续导致能源价格高企的时间不确定性
2. 霍尔木兹海峡长期紧张可能导致的柴油供应风险
3. 商品价格和汇率的波动性
4. 政治、财务和运营风险
5. 天气因素对生产的影响
6. 中东地区冶炼厂关闭后重启可能需要6-12个月的时间
