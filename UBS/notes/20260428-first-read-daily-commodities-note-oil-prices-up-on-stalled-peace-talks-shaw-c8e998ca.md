## 核心观点 / Key Takeaways
1. 锂价前景上调：瑞银上调锂价预期，主要基于储能系统(ESS)日益乐观的展望、中东冲突推动电动车需求增长以及中国电动卡车需求加速。
2. IGO产量不及预期：IGO旗下Greenbushes矿山产量为35.1万吨，低于市场预期，主要因品位下降和回收率减弱，导致全年产量指引下调。
3. 锂市场趋紧：随着市场趋紧，瑞银对锂价持更乐观态度，上调锂长期价格预期至每吨1,400美元，并上调相关公司目标价。

## 关键数据 / Key Data
- 铁矿石价格：61%品位，中国CFR价格，+0.2%，每吨108美元
- 布伦特原油：+2.7%，每桶108美元；WTI原油：-1.9%，每桶97美元
- 锂价上调：锂辉石价格预期上调23%（短期）和17%（长期）至每吨1,400美元(SC6.0)
- IGO目标价：上调至每股9.75澳元，维持买入评级
- Mineral Resources目标价：上调至每股73澳元，维持买入评级

## 投资建议 / Investment Recommendation
- IGO (IGO.AX)：买入评级，目标价A$9.75
- Mineral Resources (MIN.AX)：买入评级，目标价A$73.00

## 风险提示 / Risk Factors
1. 大宗商品价格和汇率波动可能显著不同于预期
2. 资源行业面临政治、财务和运营风险，可能显著影响公司/行业表现
3. 中东冲突导致的全球能源危机可能带来更广泛的不确定性
4. 采矿成本通胀，特别是柴油价格相关成本上升
