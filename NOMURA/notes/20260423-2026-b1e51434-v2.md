## 报告主体 / Report Subject

## 核心观点 / Key Takeaways
1. 2026年一季度业绩超预期：公司营收和净利润分别同比增长117.8%和79.2%，主要得益于产品出货良好及价格上涨。
2. 2025年核心产品销量增长显著：钴酸锂、三元材料、磷酸铁锂销量分别同比增长41.3%、7.9%和2170.8%。
3. 前沿材料布局取得突破：公司在全固态正极材料、聚阴离子钠电材料、磷酸锰铁锂等新型材料领域取得阶段性进展。
4. 维持增持评级，上调目标价：基于业绩增长和行业估值修复，野村上调公司目标价至103.2元，隐含29.1%上行空间。

## 关键数据 / Key Data
- 2025年营收：198.8亿元（同比+46.7%）
- 2025年归母净利润：7.5亿元（同比+41.8%）
- 2026年一季度营收：65.9亿元（同比+117.8%）
- 2026年一季度归母净利润：2.2亿元（同比+79.2%）
- 2026年一季度钴酸锂销量：1.47万吨（同比+19.8%）
- 2026年一季度动力电池正极材料销量：1.57万吨（同比+25.7%）
- 目标价：103.2元（上调前为92.25元）
- 上行空间：+29.1%
