## 报告主体 / Report Subject
野村东方国际证券研究观点每周汇总更新（2026年4月4日至2026年4月10日）

## 核心观点 / Key Takeaways
1. **电子行业**：Q2存储合约价继续大幅上升，小米部分机型提价。看好AI算力链持续增长，特别是PCB/CCL、光模块板块；同时关注存储芯片和MLCC等具备涨价潜力的领域。生益科技(600183.SH)、胜宏科技、中际旭创、江海股份为推荐标的。

2. **汽车行业**：均胜电子(600699.SH)业绩符合预期，在智能网联领域聚焦光模块等新技术，在手订单充沛，盈利能力稳健。MLCC市场整体需求可能比日本出口贸易数据显示得更好。

3. **电力设备行业**：继续看好锂电池材料的投资机遇。福斯特(603806.SH)有望受益于胶膜价格修复和电子材料突破，2026年胶膜价格修复中，电子材料有望逐步成长为第二增长。

4. **食品饮料行业**：贵州茅台(600519.SH)小幅上调飞天销售合同价和自营零售价，向市场传递价值坚挺信号，逐步重掌市场定价权。海天味业(603288.SH)主业韧性凸显，盈利持续改善。

5. **医药行业**：礼来以最高约78亿美元对价收购Centessa，核心意图是前瞻布局食欲素受体2(OX2R)激动剂赛道，切入中枢神经系统睡眠-觉醒调控这一高壁垒细分领域。

## 关键数据 / Key Data
1. **均胜电子(600699.SH)**：2025年实现营收611.83亿元，同增9.52%；归母净利润13.36亿元，同增39.08%。维持增持评级和目标价36.24元不变。

2. **江海股份(002484)**：2025年实现营收54.8亿元，同比增长14.1%；归母净利润6.7亿元，同比增加3%。上调目标价至39元，维持增持评级。

3. **福斯特(603806)**：2025年实现营收154.9亿元(同减19.1%)，归母净利润7.7亿元(同减41.1%)。维持增持评级，上调目标价至25.65元。

4. **海天味业(603288)**：2025年实现营收288.73亿元，同比提升7.32%；归母净利润70.38亿元，同比提升10.95%。维持增持评级，下调目标价至49.36元。

5. **安井食品(603345)**：维持增持评级，上调目标价至118.18元，隐含20.8%的上行空间。

## 投资建议 / Investment Recommendation

- **推荐关注**：生益科技(600183.SH)、胜宏科技、中际旭创、兆易创新、三环集团、风华高科、徐工机械(000425.SZ)、三一重工(600031.SH)、芯碁微装(688630.SH)
